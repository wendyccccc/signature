# Vue3手寫板

### 重點
- 改善筆跡鋸齒狀
- 簽名時的線條必須平滑
- 手持Apple Pencil時不會偵測到手指觸摸事件
- 簽名處隨著螢幕撐滿螢幕
- 能清除畫跡、能存檔成png、能存成base64

### 其他套件
- 搭配element plus開發，但其實不用這個也可以

## Project setup
```
npm install
```

### Compiles and hot-reloads for development
```
npm run serve
```

### Compiles and minifies for production
```
npm run build
```

### Lints and fixes files
```
npm run lint
```
